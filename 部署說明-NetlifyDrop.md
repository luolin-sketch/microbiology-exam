# Netlify Drop 一鍵部署說明

## 最簡單的部署方式（5分鐘完成）

### 步驟一：準備檔案
1. 下載並解壓縮 `microbiology-exam-deploy.zip`
2. 您會看到三個檔案：
   - `index.html`
   - `favicon.ico`
   - `assets` 資料夾

### 步驟二：上傳到Netlify Drop
1. 訪問：**https://app.netlify.com/drop**
2. 將整個資料夾（或選中所有檔案）**直接拖放**到網頁上的虛線框中
3. 等待上傳完成（約10-30秒）

### 步驟三：獲得永久網址
上傳完成後，Netlify會立即顯示您的網站網址，格式如：
```
https://random-name-123456.netlify.app
```

這個網址是**永久有效**的，您可以：
- 立即訪問和使用
- 分享給學生
- 在Netlify帳號中管理（可選）

### 進階選項（可選）
如果您註冊Netlify帳號，可以：
- 修改網址名稱（如：`microbio-exam.netlify.app`）
- 綁定自訂域名
- 查看訪問統計
- 更新網站內容
